# Affan xp Portfolio

A nostalgic, high-fidelity Windows XP-inspired personal portfolio website. This project replicates the iconic look and feel of the Windows XP operating system from the early 2000s, featuring a fully functional window management system, authentic boot sequence, and interactive desktop environment.

## 🚀 Features

- **Authentic Boot Sequence**: Complete with the classic Windows XP logo, progress bar, and login screen.
- **Window Management System**: Multi-window environment with minimize, maximize, and close functionality.
- **Interactive Desktop**: Classic desktop icons, selectable states, and single-click support for mobile.
- **Start Menu**: Fully functional XP-style Start Menu for quick navigation.
- **Internet Explorer Integration**: View project details through a themed browser interface with authentic loading animations.
- **Retro Sound Effects**: Integrated classic startup and shutdown sounds.
- **Mobile Optimized**: Responsive design that adapts the XP experience for smaller screens.

## 🛠️ Tech Stack

- **Core**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) & Vanilla CSS
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **Window Logic**: [React Rnd](https://github.com/bokuweb/react-rnd)
- **Icons**: Custom SVG components based on classic XP assets.
- **Build Tool**: [Vite](https://vitejs.dev/)

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
